# Task Manager

A simple fullstack task manager app using **Node.js**, **Express**, and a local JSON file for storage.

## Features
- Add tasks
- Delete tasks
- View all tasks

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `node server.js`
4. Open browser at `http://localhost:3000`
